# davicosas

Este repositorio está pensado para documentar problemas de trabajo y resolverlos con apoyo de Copilot de forma clara y práctica.

## Cómo usarlo

Cuando subas un problema, incluye:

1. **Contexto**: qué sistema/proceso está fallando.
2. **Síntoma exacto**: error, comportamiento esperado vs real.
3. **Evidencia**: logs, pasos para reproducir, ejemplos.
4. **Restricciones**: tiempos, tecnología, límites de seguridad.

Con esa información, Copilot puede ayudarte a:

- Analizar la causa raíz.
- Proponer una solución concreta.
- Implementar cambios mínimos y seguros.
- Validar que la solución realmente funcione.
